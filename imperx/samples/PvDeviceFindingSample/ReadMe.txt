========================
Finding devices
========================

This sample code illustrates how to use the PvSystem, PvInterface, PvDeviceInfo and PvDeviceFinderWnd classes to detect and enumerate IP Engines.

1. Introduction

Using the PvSystem and PvInterface class, this sample shows how to:
 * Find IP Engines on a network

Using the PvDeviceFinderWnd class, this sample shows how to:
 * Find IP Engines on a network using the GUI

2. Prerequisites

This samples assumes that:
 * You have a GigE Vision Device connected to a network adapter.

3. Description

3.1 PvDeviceFindingSample.cpp

Sample code as described above. Please refer to the comments in the source code for information on the methods used in this sample.
