========================
Using PvBufferWriter
========================

This sample code illustrates how to use the PvBufferWriter and PvBuffer classes.

1. Introduction

Using the PvBuffer class, this sample shows how to:
 * Create and allocate a buffer, and fill the buffer with data

Using the PvBufferWriter class, this sample shows how to:
 * Save the Buffer as a bitmap file

2. Prerequisites

This samples assumes that:
 * You have a GigE Vision Device connected to a network adapter.

3. Description

3.1 PvBufferWriterSample.cpp

Shows how to use the PvBufferWriter and PvBuffer classes. Please refer to the comments in the source code for information on the methods used in this sample.
