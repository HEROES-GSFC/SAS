Detailed Description

Counters and statistical properties of a PvStreamRaw object.

PvStatistics provides access to the cumulative streaming properties collected while receiving data from a GigE Vision transmitter. It provides the same information that can be accessed through PvStream::GetParameters but in a straight C++ interface instead of a dynamic GenICam interface.
