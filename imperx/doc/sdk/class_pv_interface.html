Detailed Description

Represents one interface (network adapter) on a system (the PC).

PvInterface lets you find all GigE Vision devices accessible by a single NIC on your PC.

To use this class, see PvSystem.

PvUInt32 PvInterface::GetDeviceCount (  )  const

Get the number of found GigE Vision devices on this PvInterface (NIC level).

The GetDeviceCount method reports the number of GigE Vision devices found by a single NIC on your PC. To search for all reachable GigE Vision devices on the system (PC), use PvSystem::Find. To search for all reachable GigE Vision devices on an interface (network adapter), use PvInterface::Find.

Returns:
The number of GigE Vision devices found by through this PvInterface. A return of 0 suggests either no GigE Vision devices, or the Find method wasn't called first.

bool PvInterface::IsUsingSubnetLimitedBroadcasts (  )  const

Returns true if subnet limited broadcasts are used to discover devices.

On Linux regular (non root) users can only send subnet limited broadcasts. This prevents devices not on the same subnet as the network interface to be found as expected.

If being able to find devices on a different subnet (including 0.0.0.0 default IP configuration) on Linux is required, the application using PvSystem and PvInterface must be run with root privileges.

On Windows, this limitation currently does not exist. This property is always expected to evaluate to true.

Returns:
True if subnet limited broadcasts are used to find devices.
