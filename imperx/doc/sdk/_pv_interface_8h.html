Enumeration Type Documentation

enum PvInterfaceType

The type of driver that would be leveraged for streaming data on this interface.

Enumerator:
PvInterfaceNetworkStack 

No eBUS driver was enabled on this interface. In this case, you should still be able to send and receive data from this interface but with reduced performance.

PvInterfaceEBus 

An eBUS driver is enabled on this interface. An eBUS driver increases streaming performance by bypassing part of the network stack and minimizing system CPU usage.
