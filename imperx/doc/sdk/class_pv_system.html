Detailed Description

Find interfaces (network adapters) and GigE Vision devices reachable from this PC.

PvSystem finds all interfaces (network adapters) and returns PvInterface objects. PvInterface finds all GigE Vision devices on a specific interface and returns PvDeviceInfo objects. PvDeviceInfo provides the information required to connect to a GigE Vision device.

To connect to the GigE Vision device, see PvDevice. To bring up a user interface that allows one to select a GigE Vision device, refer to PvDeviceFinderWnd.

For a complete example illustrating how to discover and connect to a GigE Vision device, refer to the sample entitled PvDeviceFindingSample.

PvResult PvSystem::Find (  ) 

Find all interfaces and GigE Vision devices reachable from this system (the PC).

The PvSystem::Find and PvInterface::Find methods search for all GigE Vision devices on a network. The PvSystem version searches using all interfaces on the system; the PvInterface version, only one interface. The former works as if making multiple, simultaneous calls to the latter, albeit concurrently.

The methods are blocking and won't return for the duration set using SetDetectionTimeout.

PvResult PvSystem::SetDetectionThreadsPriority ( PvUInt32  aPriority  ) 

Set the priority of the threads used to search the devices.

Each interface has one thread that is used to asynchronously discovery devices. Only in use when searching devices using PvSystem::Find or PvInterface::Find.

If you change the thread priorities, it may alterate the stability of the system. For this reason, if you report a bug to Pleora, ensure to clearly mention that you change threads priorities.

See GetDetectionThreadsPriority

void PvSystem::SetDetectionTimeout ( PvUInt32  aTimeout  ) 

Set the maximum time to search the system for interfaces and GigE Vision devices.

The maximum time the PvSystem::Find and PvInterface::Find methods wait for responses from GigE Vision devices before returning.

Parameters:
[in] aTimeout The maximum time to search for GigE Vision devices, in ms. The default value of 1500 is generally a good value in most situations. Values beyond 100 to 60000 are set to 100 and 60000, respectively.
