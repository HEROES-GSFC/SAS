Detailed Description

Raw data interface to a PvBuffer.

A PvBuffer represents a generic buffer with no specific payload type. If the payload type is raw data (according to the GigE Vision specification), the user can obtain a pointer to a PvRawData buffer interface through PvBuffer::GetRawData.

This PvRawData pointer is owned by the PvBuffer and does not need to be released after use. It is valid as long as the PvBuffer exists.
