Detailed Description

Receive data from a GigE Vision transmitter (data receiver properties dynamically accessible using a PvGenParameterArray interface).

This class inherits from PvStreamBase which provides a means of opening a connection to the data receiver and obtaining data from it. For further information on how PvStream fits in with the process of receiving data from a GigE Vision transmitter, refer to the page entitled Receiving Data from a GigE Vision Transmitter.
