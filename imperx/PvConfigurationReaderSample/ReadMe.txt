========================
Using PvConfigurationReaderSample
========================

This sample code illustrates how to use the PvConfigurationReader and PvConfigurationWriter classes.

1. Introduction

Using the PvConfigurationWriter class, this sample shows how to:
 * Store different kind of configuration to a pvxml file
 * Store a device configuration with a name
 * Store a stream configuration with a name
 * Store a string information with a name

Using the PvConfigurationReader class, this sample shows how to:
 * Open and restore device and stream configuration
 * Restore the string information

2. Prerequisites

This samples assumes that:
 * You have a GigE Vision Device connected to a network adapter.

3. Description

3.1 PvConfigurationReaderSample.cpp

Shows how to use the PvConfigurationReader and PvConfigurationWriter classes. Please refer to the comments in the source code for information on the methods used in this sample.
